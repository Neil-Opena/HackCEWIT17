Array Problem Set

  • printReverse() - take an array as an argument and print out elements in the array in reverse order (don't reverse actual array)
  • isUniform() - takes an array as an argument and returns true if all elements in the array are identical
  • sumArray() - takes an array as an argument and returns the sum of all the numbers in the array
  • max() takes an array as an argument and return the maximum number in the array
